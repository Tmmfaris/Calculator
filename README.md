# Calculator Web Application

A simple and responsive **Calculator Web App** built using **HTML, CSS, and JavaScript**.
This project performs basic arithmetic operations and demonstrates fundamental **front-end development concepts such as DOM manipulation, event handling, and responsive UI design**.

```
cd Calculator
```

### Open the project

Open **index.html** in your browser.

---

# Deployment (GitHub Pages)

This project is deployed using **GitHub Pages**.

### Steps to deploy

1. Go to **Repository Settings**
2. Click **Pages**
3. Under **Source**, select:

```
Deploy from a branch
```

4. Choose:

```
Branch: main
Folder: /(root)
```

5. Click **Save**

Your site will be available at:

```
https://tmmfaris.github.io/Calculator/
```
